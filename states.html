Analysis of State Websites!

1. Kansas

The state website for Kansas provides users with an easy to read site that is simple and straight
to the point. The colors chosen for the site compliment each other nicely, giving users a friendly
welcome when they first open the site. Not only that, but the website gives clear information to the
help center and quick access tools with a helpful navigation bar at the top of the screen.

See more about Kansas's state website HERE.

2. Alaska

While Alaska's website may not surmount the aspects of Kansas's, this site incorporates a mountainous
theme that helps the user feel as if they are surrounded by the state's beauty at home on their own
computer. The tables and dropdown menus used on the site are not super clean-cut and simple, but the
website loads very quickly overall and provides users with an easy-to-use site on mobile devices.

See more about Alaska's state website HERE.

3. Maine

Maine's state website contains a lot of useful information, but the styling doesn't follow a consistent
theme and can be distracting in that way. There isn't a super logical order of information and this fails
to really hook a user. Aside from these interruptions, Maine's state website gets the job done with giving
enough information to satisfy an online user through a good use of lists and separated, boxed information.

See more about Maine's state website HERE.

4. Arkansas

I love the themes used in this website, but it is just too darn slow! I think the developers went a little
too overboard on styling causing the website to slow down, but everything else looks great. Well-organized
information is separated logically in different dropdown menus and links, along with a useful and well-built
search bar at the top for easy access to information that might be hard to find. Overall, the slowness of
the website puts this in the #4 spot.

See more about Arkansas's state website HERE.

5. Maryland

Maryland's state website is not like the others in that it is a simple, one page site with not a lot of
scrolling to do on the main page. Looking at this from a computer is not very pleasing. The lack of dropdown
menus and external links makes it tough to find information quickly. If this site had the searchbar that
Arkansas's had, it would be much more functional. Although it has its flaws on a computer screen, this site
works very well for a mobile device.

See more about Maryland's state website HERE.
